Omeka Multiinstanz Manager - OMIM¶

OMIM ermöglicht das Bereitstellen und die Verwaltung von Omeka Instanzen. Es setzt dabei eine Produktionslinie von min. zwei Servern voraus.

  • Entwicklungs- bzw. Redaktionsserver (Ausstellungsredaktionssystem)
  • Produktions- bzw. Liveserver (Mutterinstanz) bzw. Produktions-Ausspielungsserver mit 1-n Servern (für Loadbalancing)

OMIM wird auf dem Redaktionsserver installiert und eingerichtet. Die Software ermöglicht das Veröffentlichen von Instanzen auf den Ausspielungsservern.
